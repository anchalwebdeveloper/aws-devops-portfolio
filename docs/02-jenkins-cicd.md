# Automated CI/CD Pipeline on AWS EC2

## Overview
Configured a fully automated build and deployment pipeline enabling rapid and error-free releases.

## Architecture Diagram
![Jenkins CI/CD Pipeline](./assets/cicd.png)

## Responsibilities
- Installed & configured Jenkins on EC2
- Integrated Jenkins with GitHub using webhooks
- Implemented Jenkins Pipeline (Jenkinsfile)
- Automated deployment to production EC2 instance
- Configured rollback & version control strategy
- Enabled zero-downtime releases using NGINX reload

## Tools Used
GitHub | Jenkins | EC2 | Linux | Shell Scripting

## Outcome
- Deployment time reduced from 30 mins to <5 mins
- Eliminated manual deployment errors
- Enabled continuous delivery
