# Docker-based Application Deployment

## Overview
Containerized and deployed a lightweight and scalable web application using Docker.

## Architecture Diagram
![Docker Deployment Workflow](../assets/docker.png)

## Responsibilities
- Built custom Dockerfile for app environment
- Published container images on Docker Hub / ECR
- Pulled and deployed container on EC2 using docker-compose
- Configured persistent storage & container restart policies
- Hardened deployment using IAM roles and private ECR access

## Tools Used
Docker | Docker Hub/ECR | EC2 | Linux

## Outcome
- 100% environment consistency across dev & prod
- Faster deployment and simplified scaling
